# Blazor · DiskMap, the shared source

This is the shared source project of the DiskMap sample, the C# the host and the front end both compile, and not an
app of its own. It holds the DTOs, the size formatting and the source-generated JSON serializer.

See [AOTrino.Samples.Blazor.DiskMap](../AOTrino.Samples.Blazor.DiskMap/README.md) for what the sample does and why
this is a shared source project rather than a referenced assembly.
