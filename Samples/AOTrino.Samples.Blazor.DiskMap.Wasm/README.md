# Blazor · DiskMap, the front end

This is the Blazor WebAssembly front end of the DiskMap sample, and not an app of its own. It is built and embedded
by the host project, which the solution builds instead of this one.

See [AOTrino.Samples.Blazor.DiskMap](../AOTrino.Samples.Blazor.DiskMap/README.md) for what the sample does and how
the three projects fit together.
